Git is a VCS (Version Control System) which allows to easily track and navigate through code history.

::::tip[Basic commands you should know]
- `git status` - shows current status (working change)
- `git pull` - collects changes from the remote repository
- `git push` - publish your committed changes to the remote repository
- `git add` - adds changes to commit index
- `git commit` - saves index (added changes) into repository
- `git merge` - applies changes between branches (join them)
- `git rebase` - reapplies changes on top of other branch
::::
