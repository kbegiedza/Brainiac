## Customization

### Custom colors

Define custom colors inside `tailwind.config.js`

```js
export default {
    theme: {
        extend: {
            colors: {
                'brand-background': '#1CA4FF'
                'brand-accent': '#92DEFF',
            }
        }
    }
    // ...
}
```
